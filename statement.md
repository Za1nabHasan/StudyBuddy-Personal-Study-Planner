StudyBuddy – Project Statement 

# Problem Statement 
Students often struggle to keep track of multiple subjects, deadlines, and study tasks. With no centralized 
place to manage academic workload, tasks get delayed or forgotten. StudyBuddy solves this through a 
lightweight, easy-to-use task management system. 

# Scope of the Project 
StudyBuddy focuses on everyday academic planning. 
It includes adding, viewing, searching, sorting, updating, and deleting study tasks. 
It uses Python and basic file handling to demonstrate real-world application of programming concepts 
taught in Semester 1. 

# Target Users 
• School and college students 
• Anyone who wants a simple study scheduling tool 

# High-Level Features 
1. Add new study tasks 
2. View task list 
3. Search tasks by subject 
4. Sort tasks by deadline 
5. Mark tasks as completed 
6. View pending vs completed count 
7. Delete tasks 
8. Persistent storage using file handling
